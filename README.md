## 如何使用 | Usage

直接**复制粘贴**到你的目录即可。

Just **Copy them** and **Paste into** your workspace.

## 如何测试 | Test

使用 GCC 编译器编译：

Compile with GCC compiler:

```bash
$ g++ ./BiTree.cpp -o ./build/BiTree.exe -Wall
```

然后运行：

Then exec it:

```bash
$ ./build/BiTree.exe
```

你会看到一个**假的不能再假**的用户界面，然后就可以开始测试了。

You will see a **fake** User Interface and then you can use it for you "test".
